1.1 Primary Logo Lockups

This is the primary Airbrake.io logo to be used in the majority of marketing and promotional cases. For special cases, e.g. limited print colors, see Section 1.4

Colors Used
• Background HEX value is #f58c54
• HEX value for pane’s left and middle areas as well as the flap on the right is #ffffff
• Pane’s right area HEX value is #e4e4e4
• “Airbrake.io” HEX value is #ffffff

Typeface Used
• Typeface used for ”A” in the icon is Museo 700
• Typeface used for “Airbrake.io” logotype is Museo 300

1.2 Other Possible Logo Lockups

Icon-only version of the logo can be used when there’s not enough space to fit the primary logo and shrinking it down results in unreadable text. If there's not enough horizontal space, see vertical logo lockup.

“Aibrake.io” logotype can be used on its own, without an icon. Typeface used is Museo 700. 

Logotype should be used on an area with 3 heights horizontal margin and 1.5 vertical margin.

Sometimes a vertical composition is required but space allows for both icon and logotype to be used. In this case it's advised to use the vertical logo lockup.

1.3 Allowed Colour Schemes

Airbrake logo uses a sophisticated colour system that allows using it with virtually any colour. Due to the logo’s simplicity, colours used play a major role for Airbrake brand so it is strongly advised to use classic Airbrake colour schemes at all times.

Using various colours:
• White on orange – preferred
• Orange on white – preferred
• White on black – allowed
• White on colour other than orange – not advised
• Colour on black – prohibited

Using white Airbrake logo is okay on virtually every contrasting color (no light shades of grey, etc.). It is, however, to be avoided in favor of an Airbrake-orange background whenever possible.

One thing to remember when using white logo on a coloured background is that A inside the icon should be the same colour as the background.

Orange Airbrake logo is not to be used on any background other than white.

Airbrake.io has its own, distinguished gradient. At times using the gradient is required. It can only be used with the logotype lockup (no icon), for other gradient use cases see section 1.4

Colours used for linear gradient are:
• Top or right – #ff8e52
• Bottom or left – #fba731

Radial or any other gradients are prohibited. Use of Airbrake logo on any other gradients is prohibited.

1.4 Special Cases

Single Colour Lockup
Whenever circumstances don’t allow use of more than one colour, this single colour lockup is to be used.

Social Media Icons
Due to proportional limitations of squared images used in most of social media sites (Twitter, Facebook, Github, etc.), a separate social media icon was designed.

”A” on a gradient is the primary version and to be used at all times unless otherwise specified.

Using a Tagline
When a tagline is needed, no additional changes are made to the primary lockup. The tagline is added, typeface and colour used are Museo 300, #999999
